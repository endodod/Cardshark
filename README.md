# CARDSHARK v1.0

> You found an old laptop. You booted it up. The dealer has been waiting.

A retro terminal-style blackjack game with casino mini-games, built as a
personal project to explore frontend game development.

## Features

- **Blackjack** with slot machine modifiers — spin before each hand to
  change the rules in your favour (or against you)
- **Dice Challenge** — risk your round winnings on a dealer-offered roll
- **Roulette Strip** — bet on a scrolling number strip with live dealer narration
- **Texas Hold'em Standoff** — heads-up poker duel with three reveal variants
- **CRT aesthetic** — scanlines, vignette, typewriter dialogue, VT323 font
- **Procedural sound** — all audio generated via Web Audio API, no files

## Stack

- [Next.js 14](https://nextjs.org) — App Router
- JavaScript
- [Tailwind CSS](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion) — animations
- Web Audio API — procedural sound engine

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
